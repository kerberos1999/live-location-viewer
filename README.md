# live-location-viewer

## requests from server to client
| index | description |
|---|---|
| 0 | return client array |
| 1 | remove player |
| 2 | normal message |
| 3 | start game |
| 4 | position request |
| 5 | position update |

## requests from client to server
| index | description |
|---|---|
| 0 | request client array |
| 1 | start game |
| 2 | close game |
| 3 | provide position |
