# live-location-viewer

## requests from server to client
| index | description |
|---|---|
| 0 | new player |
| 1 | normal message |
| 2 | position update |
| 3 | position request |

## requests from client to server
| index | description |
|---|---|
| 0 | new player with role for client |
| 1 | start game |
| 2 | close game |
